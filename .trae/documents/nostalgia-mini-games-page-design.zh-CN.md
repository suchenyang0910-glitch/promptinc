# 页面设计说明（桌面优先 + 移动端可玩）

## 全局设计（适用于所有页面）

### Layout
- 桌面优先：内容容器 `max-width: 1200px` 居中；两侧留白。
- 移动端：断点 `<=768px` 自动改为单列；游戏交互区优先占据首屏。
- 主要布局：页面整体用 CSS Grid（顶部导航/主体/底部），主体内部按模块用 Flexbox。

### Meta Information（全站默认）
- Title 模板：`{页面名} | 怀旧小游戏`
- Description：一句话说明“9 款即点即玩怀旧小游戏 + 排行榜”。
- Open Graph：`og:title/og:description/og:type=website`；分享图可用站点默认图（占位）。

### Global Styles（Design Tokens）
- 背景色：`#0B1220`（深色）
- 卡片底色：`#111B2E`；边框：`rgba(255,255,255,0.08)`
- 主色/强调色：`#7C3AED`（按钮与高亮）
- 文字：主文 `#E5E7EB`，次文 `#9CA3AF`
- 字体：系统字体栈；字号：12/14/16/20/24
- 按钮：主按钮圆角 10px；hover 提亮 6%；disabled 降低不透明度
- 链接：下划线仅 hover 出现；避免影响游戏区域误触

### 广告位占位（统一组件 AdSlot）
- 位置：
  - 顶部横幅：导航下方，固定高度 90px（桌面）/ 60px（移动）
  - 底部横幅：页面底部上方 90px/60px
- 交互：纯占位不可点击（或仅记录曝光），不覆盖游戏关键按钮；移动端底部横幅在游戏进行中可自动折叠为 32px。

## 页面 1：首页（游戏大厅）

### Meta
- Title：`怀旧小游戏大厅 | 怀旧小游戏`
- Description：突出“9 款小游戏 + 无需下载 + 可上榜”。

### Page Structure
- 纵向堆叠：导航栏 → 顶部广告位 → 游戏矩阵 → FAQ → 底部广告位 → 页脚。

### Sections & Components
1. 顶部导航（NavBar）
   - 左侧 Logo（返回首页）
   - 右侧：`排行榜` 入口
2. 顶部广告位（AdSlot/Top）
3. 游戏矩阵（GameGrid）
   - 桌面：3 列卡片网格；移动：1 列大卡
   - 卡片元素：封面占位图、游戏名、简介、标签（休闲/反应/益智）、CTA“开始游戏”
4. FAQ 摘要（FAQAccordion）
   - 3-5 条站点级问题，默认折叠；支持“去排行榜看看”内链
5. 底部广告位（AdSlot/Bottom）+ 页脚（版权/免责声明占位）

### Responsive
- 移动端卡片 CTA 固定在卡片底部，避免误触。

## 页面 2：游戏页面（/games/[slug]）

### Meta（每游戏独立）
- Title：`{游戏名} 在线玩 | 怀旧小游戏`
- Description：`{一句话玩法 + 可提交分数上榜}`
- FAQ：该游戏 3-6 条常见问题（例如“怎么玩/计分规则/是否支持手机/如何上榜”）。

### Page Structure
- 桌面：左右结构（左：游戏区；右：信息与榜单预览）
- 移动：上下结构（上：游戏区；下：操作/提交/榜单入口）

### Sections & Components
1. 顶部导航 + 顶部广告位
2. 游戏区（GameStage）
   - 容器：16:9 或自适应高度（移动端优先首屏）
   - 状态：未开始/进行中/暂停/结束
   - 控件：开始、暂停、重开（按钮靠边，不遮挡主视野）
3. 触控层（TouchControls，仅移动端显示）
   - 依据游戏类型切换：方向键/单按钮/滑动提示
   - 交互态：按下高亮、震动（可选）
4. 成绩区（ScorePanel）
   - 显示：本局分数、最佳分数（本设备）
   - 提交：昵称输入（可选，限制长度）+ “提交分数”按钮
   - 成功态：提示“已上榜/查看排行榜”并提供 CTA
5. 右侧/下方：排行榜入口（LeaderboardTeaser）
   - 展示该游戏 Top 5 预览 + “查看完整榜单”
6. SEO 内容区（GameFAQ）
   - FAQ 折叠组件；默认展示 1 条
7. 底部广告位

### Responsive & Performance
- 移动端游戏区应避免页面滚动干扰：进行中可锁定 body 滚动。
- 游戏渲染优先 60fps；资源延迟加载；离开页面自动暂停。

## 页面 3：排行榜页面（/leaderboard）

### Meta
- Title：`排行榜 | 怀旧小游戏`
- Description：突出“全站榜/单游戏榜切换”。

### Page Structure
- 纵向堆叠：筛选条 → 榜单表格/列表 → 闭环 CTA。

### Sections & Components
1. 顶部导航 + 顶部广告位
2. 筛选条（FilterBar）
   - Tab：`全站` / `按游戏`
   - 下拉：选择 game（9 个 slug）
   - 排序：默认按分数降序（仅展示必要字段）
3. 榜单（LeaderboardList）
   - 桌面：表格（排名/昵称/分数/时间）
   - 移动：列表卡片（排名徽标 + 昵称 + 分数）
4. 我的成绩（MyBest）
   - 若能匹配 device_id，则展示“我的最好成绩”与所在区间提示（可选）
5. 闭环 CTA（LoopCTA）
   - `再玩一次`（跳回当前筛选的游戏）
   - `回大厅换游戏`
6. 底部广告位

### Interaction
- 从“提交成功”跳转而来时：自动选择对应游戏筛选，并滚动到榜单顶部。
